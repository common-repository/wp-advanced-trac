== Description ==
Advanced version of the Wp-trac plugin http://www.johnciacia.com/wp-trac/ with some new features, code fixes.
First of all, it was developed to show roadmaps of your projects to other users.

Main changes between original version:
* Project end date is not required;
* Use in [wptrac project_id_number ] in your posts,pages to show project roadmap to your users, for example [wptrac 1];
* The last date of the current tasks is shown as possible release date;
* Future tasks are not displayed to users.

Example and plugin roadmap is here http://eeexception.org/my-projects/wp-advanced-trac/

== Installation ==

1. Unpack archive and upload directory `wp-advanced-trac` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[wptrac project_id_number]` in your pages,posts (for example [wptrac 1] for the project with id 1)
